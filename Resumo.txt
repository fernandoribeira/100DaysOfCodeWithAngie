- Configurações VSCode

    Mudar idioma

        Pressione Ctrl + Shift + P
        Digite "Configure Display Language"
        Selecione "Configure Display Language"
        Selecione o idioma "português (Brasil)"

    Extensions

        Auto Close Tag
        Auto Rename Tag
        Color Highlight
        Image Preview
        Prettier - Code formatter
        Live Server
        vscode-pdf

    Settings

        Auto Save set to "onFocusChange"
        Default Formatter set to "Prettier - Code formatter (esbnp.prettier-vscode)"
        Format on Save set to "true"
        Word Wrap set to "on"
        Tab Size set to "4"
        Tab Width set to "4"
        Multi Cursor Modifier set to "ctrlCmd"

- Como criar um documento HTML básico no VSCode

    html:5 + Enter

- Carregar código JavaScript embutido no HTML

    Inserir a tag abaixo no arquivo HTML antes do fechamento da tag <body> e dentro dela o código JavaScript.

    <script>
        ...
    </script>

- Carregar código JavaScript separado do HTML

    Criar um arquivo com extensão .js separado e adicionar a tag abaixo no arquivo HTML indicando seu nome e caminho antes do fechamento da tag <body>.

    <script src="script.js"></script>

- Escrevendo no console

    O comando console.log() é uma função do JavaScript usada para exibir informações no console do navegador, sendo muito útil para depuração e acompanhamento da execução do código.

    Exemplo:

    console.log("Olá mundo");

    Para contatenar informações utilizamos a virgula "," ou o simbolo de soma "+".

    A "," aplica um espaço entre as informações ja o "+" não aplica, sendo necessario aplicar um espaço manualmente entre as informações.

    Exemplo:

    console.log("Olá", "mundo");

    console.log("Olá" + " " + "mundo");

- Uso de aspas duplas

    Utilizada para criar campos de texto chamdos "strings", por padrão é utilizada a aspas duplas cercando o texto conforme exemplo abaixo:

    const nome = "Fernando";

- Uso de ponto e virgula

    O ponto e vírgula indica o fim de uma instrução, exmeplo:

    let a = 10;

- Palavras reservadas

    São palavras que pertencem à linguagem, elas possuem significado próprio e não podem ser usadas como identificadores de nenhum tipo, por exemplo as palavras abaixo:

    if
    new
    let

- Variaveis e constantes

    Uma variável é um espaço na memória utilizado para armazenar um valor.

    Declaração

        let - Forma moderna e recomendada quando o valor pode mudar, pode ser declarada vazia.
        const - Usada quando o valor não pode mudar (tipos comuns), obrigatoriamente tem de ser atribuido um valor ao ser declarada

    Convenção de nomes

        Não pode começar com número
        Não pode conter espaços
        Não pode usar palavras reservadas
        Pode usar letras, números, _ e $
        Usar padrão "camelCase", primeira palavra minúscula, próximas com inicial maiúscula.
        Constantes devem ter todas a letras em maiúsculo quando o valor é realmente fixo e imutável, algo que representa uma constante "global" ou  de configuração porem pode ser escita como variavel comum em casos comuns

    Observações importantes:

        const é a forma recomendada por padrão para declarar quaisquer tipo de variáveis atualmente
        const não torna o valor imutável para objetos / arrays, apenas impede a reatribuição da variável

    Exemplos:

        let idade = 20;
        const nome = "Fernando";
        const PI = 3.1415;

- Tipagem dinamica

    JavaScript é uma linguagem de tipagem dinâmica, ou seja:

    Você não define o tipo da variável
    O tipo é definido automaticamente pelo valor atribuído
    Uma variável pode mudar de tipo durante a execução

- Tipos de valores

    Os valores são divididos em duas grandes categorias: tipos primitivos e objetos.

    Tipos primitivos são valores simples, guardando apenas uma informação, por exemplo:

        const nome = "Maria";
        const idade = 25;

    Objetos armazenam coleções de dados e podem conter propriedades e métodos, guardando diversas informações, por exemplo:

        const pessoa = {
            nome: "Maria",
            idade: 25,
            falar() {
                console.log("Olá!");
            }
        };

- Tipos de dados

    number - Representa números inteiros e decimais
    string - Texto entre aspas simples, duplas ou crase
    boolean - Valores lógicos
    undefined - Variável declarada, mas sem valor atribuído
    null - Representa ausência intencional de valor
    symbol - Usado para criar identificadores únicos (mais avançado)
    bigint - Números inteiros muito grandes
    object - Usado para agrupar dados e funcionalidades em uma única estrutura, ele representa coleções de pares chave > valor

    /*

- Operador "typeof"

    O operador typeof é usado para verificar o tipo de um valor ou variável.

    Exemplo:

    typeof 10;           // "number"
    typeof "texto";      // "string"

- Operadores

    Aritméticos

        Usados para realizar cálculos matemáticos:

        +  adição
        -  subtração
        *  multiplicação
        /  divisão
        %  resto da divisão (módulo)
        ** exponenciação

    Atribuição

        Usados para atribuir valores a variáveis:

        =   atribuição simples
        +=  soma e atribui
        -=  subtrai e atribui
        *=  multiplica e atribui
        /=  divide e atribui
        %=  módulo e atribui
        **= exponencia e atribui

    Incrementais / Decrementais

        Servem para aumentar ou diminuir valores em 1:

        ++ incremento
        -- decremento

        Temos pré-incremento / pré-decremento:

        let i = 5;
        ++i; // i = 6

        Temos pós-incremento / pós-decremento:

        let j = 5;
        j++; // j = 6 (mas retorna 5 antes de incrementar)

        Diferença importante:

        pré: altera antes de usar
        pós: usa o valor e depois altera

    Comparadores

        Usados para comparar valores. Sempre retornam true ou false.

        ==  igual (com conversão de tipo)
        === igual estrito (sem conversão)
        !=  diferente
        !== diferente estrito
        >   maior
        <   menor
        >=  maior ou igual
        <=  menor ou igual

- Ordem dos Operadores

    A ordem define quais operações são executadas primeiro.

    Ordem simplificada (do maior para o menor):

    ()             parênteses
    **             exponenciação
    ++ --          incremento / decremento
    * / %          multiplicação, divisão, módulo
    + -            adição e subtração  
    < <= > >=      comparações
    == != === !==  igualdade
    && AND         lógico
    || OR          lógico
    = += -= *= ... atribuição
